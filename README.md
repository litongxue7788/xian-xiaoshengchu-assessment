# 西安小升初智能评估系统

基于AI技术的智能小升初评估和升学规划系统，为西安地区学生和家长提供个性化建议。

## 🚀 功能特色

- 🤖 AI智能评估与建议
- 🏫 个性化学校推荐
- 📊 可视化能力分析
- 📅 定制化时间规划
- 💬 智能对话助手

## 🛠️ 技术栈

- 纯前端HTML/CSS/JavaScript
- 阿里云百炼大模型API
- Chart.js 数据可视化
- Vercel 部署

## 🌐 在线访问

部署地址：https://xian-xiaoshengchu.vercel.app

## 📄 使用说明

1. 填写学生能力评估
2. 输入家庭基本信息
3. 查看AI生成的个性化报告
4. 获取学校推荐和升学策略

## 🎯 核心功能

### 六步评估流程
1. **能力评估** - 学业成绩、综合素养等6个维度
2. **户籍居住** - 户籍与居住地信息收集
3. **学区房产** - 入学顺位评估
4. **民办意向** - 学校偏好与预算
5. **深度需求** - 教育目标与特殊需求
6. **AI推荐** - 个性化分析与建议

### 双模式运行
- **本地模式** - 内置规则引擎，无需配置
- **AI模式** - 连接阿里云百炼，智能推荐

## 🔧 快速开始

1. 克隆项目
```bash
git clone https://github.com/your-username/xian-xiaoshengchu-assessment.git
```

2. 部署到Vercel
- 将项目推送到GitHub
- 在Vercel中导入项目
- 自动部署完成

3. 配置AI功能（可选）
- 获取阿里云百炼API Key和APP ID
- 在系统中配置API信息
- 启用AI智能推荐

## 📋 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持JavaScript
- 网络连接（AI功能需要）
