# 西安小升初智能评估系统

基于AI技术的智能小升初评估和升学规划系统，为西安地区学生和家长提供个性化建议。

## 🚀 功能特色

- 🤖 AI智能评估与建议
- 🏫 个性化学校推荐
- 📊 可视化能力分析
- 📅 定制化时间规划
- 💬 智能对话助手

## 🛠️ 技术栈

- 纯前端HTML/CSS/JavaScript
- 阿里云百炼大模型API
- Chart.js 数据可视化
- Vercel 部署

## 🌐 在线访问

部署地址：https://你的项目名.vercel.app

## 📄 使用说明

1. 填写学生能力评估
2. 输入家庭基本信息
3. 查看AI生成的个性化报告
4. 获取学校推荐和升学策略
